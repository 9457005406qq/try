# KORA 伴旅 - 面向中国推荐者的落地页

这是一个为 **KORA AI Companion** 设计的单文件落地页，目标受众是**中国人**，激发他们将产品推荐给来华的外国朋友、客户或同事。

## 🔍 项目概览

- **名称**：KORA Landing Page (CN)
- **目标**：让中国用户在 3 秒内理解 KORA，10 秒内学会推荐，并立即行动。
- **技术栈**：纯 HTML + 内联 CSS + 原生 JavaScript，零依赖，可直接在浏览器打开。
- **适配**：移动端优先，375px ~ 1024px 完美显示。

## 🗂 页面结构

| 板块 | 说明 |
|------|------|
| 通知条 | 内测提示，增强紧迫感 |
| Hero | 主标题、模拟对话界面、核心行动按钮 |
| 痛点共鸣 | 4 个中国用户常见的困扰场景 |
| 产品能力 | 6 大功能卡片（入境、支付、交通等） |
| 使用流程 | 3 步推荐引导 |
| 信任背书 | 用户真实感言 + 内测数据 |
| 适用人群 | 标签化展示目标推荐者 |
| CTA | 复制链接 & 邮箱获取双通道推荐 |
| 页脚 | 品牌信息与合规链接占位 |

## 🚀 本地使用

1. 下载或复制 `index.html` 文件到本地。
2. 双击用浏览器打开即可预览。
3. 若要部署，直接将文件上传至服务器或静态托管平台。

## 🎨 品牌定制

所有颜色通过 CSS 变量管理，在 `:root` 中修改即可更换主题：

```css
:root {
  --blue: #2563EB;   /* 主色 */
  --orange: #F97316; /* 强调色 */
}
